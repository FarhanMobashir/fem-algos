## Problem 2 : Fissures
No festivities

1.  You are given lines ((x1, y1) -> (x2, y2))
1.  You need to figure out which lines overlap and how many of them

<br />

Example input
```
0,9 -> 5,9
8,0 -> 0,8
9,4 -> 3,4
2,2 -> 2,1
7,0 -> 7,4
6,4 -> 2,0
0,9 -> 2,9
3,4 -> 1,4
0,0 -> 8,8
5,5 -> 8,2
```

<br />

#### Note
* You don't get the size of the graph created, you simply get the lines that
  are drawn and must infer it.
